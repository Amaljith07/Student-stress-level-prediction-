# 🎓 Student Stress Level Prediction

## 📌 Overview
This project focuses on predicting student stress levels using machine learning techniques. The model analyzes various factors affecting students and classifies their stress levels to help in early identification and intervention.

---

## 🚀 Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Feature engineering
- Multiple machine learning models
- Model evaluation and comparison

---

## 🛠️ Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- XGBoost
- Matplotlib, Seaborn

---

## 🤖 Models Used
- Logistic Regression
- Random Forest
- XGBoost

---

## 📊 Workflow
1. Data Collection
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Training
6. Model Evaluation

---

## 📈 Results
- Achieved good prediction performance using ensemble methods
- XGBoost provided the best accuracy among tested models
